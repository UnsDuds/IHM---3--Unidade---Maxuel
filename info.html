Como Funciona o Sistema de Busca?

Entenda os bastidores do sistema e veja o que você pode pesquisar.

Visão Geral

O sistema de busca permite que você encontre explicações rápidas sobre conceitos de HTML, CSS e JavaScript. Ele é implementado em JavaScript, utilizando uma base de dados local para responder rapidamente às consultas sem depender de servidores externos.

Como Funciona?

Passo a passo:

  1. O usuário insere um termo no campo de pesquisa.
  2. O sistema verifica se o termo está presente na base de dados.
  3. Se houver uma correspondência, o sistema exibe o termo, sua descrição e um exemplo de código.
  4. Se nenhum termo for encontrado, uma mensagem é exibida informando que não houve resultados.

Principais Componentes do Sistema

  • Base de Dados: Armazenada como um objeto JavaScript contendo categorias (HTML, CSS, JavaScript).
  • Função de Busca: Analisa o termo digitado e retorna resultados correspondentes da base de dados.
  • Interface do Usuário: Um campo de entrada e um botão para ativar a busca.
  • Exibição Dinâmica: Os resultados são exibidos de forma estilizada diretamente na página.

Lista de Termos Disponíveis

Abaixo estão os termos que você pode pesquisar no sistema:

HTML

  • div: Um contêiner genérico para organizar elementos.
  • h1: Usado para criar títulos principais.
  • a: Elemento de link para navegação.

CSS

  • color: Define a cor do texto.
  • margin: Adiciona espaço externo ao redor de elementos.
  • display: Controla o comportamento visual de elementos.

JavaScript

  • function: Declara uma função reutilizável.
  • array: Estrutura de dados para coleções.
  • event listener: Captura eventos como cliques ou teclas pressionadas.

Exemplo de Lógica de Busca

Abaixo está um exemplo do código usado para a busca:


        
    function search(term) {
      // Limpa o conteúdo anterior do contêiner de resultados para 
     exibir apenas os novos resultados da pesquisa
     resultsContainer.innerHTML = ""; 
          
     // Variável para rastrear se pelo menos um resultado foi encontrado
     let found = false;
          
     // Percorre cada categoria (html, css, javascript) na base de dados 'data'
    for (let category in data) {
     // Itera por cada item dentro da categoria atual
     data[category].forEach(item => {
     // Verifica se o termo digitado pelo usuário está incluso 
     no nome do item (case insensitive)
     if (item.term.toLowerCase().includes(term.toLowerCase())) {
     // Marca que um resultado foi encontrado
     found = true;
          
     // Cria um elemento 
 para exibir o resultado
     const resultDiv = document.createElement('div');
                  
     // Preenche o 
 com o termo, sua categoria, descrição e 
     exemplo formatados em HTML
     resultDiv.innerHTML = `
     
${item.term} (${category.toUpperCase()})

     
${item.description}

     
${item.example}

     `;
                  
     // Adiciona o 
 com o resultado ao contêiner de resultados na página
     resultsContainer.appendChild(resultDiv);
     }
     });
     }
          
     // Se nenhum resultado foi encontrado, exibe uma mensagem 
     informando isso no contêiner
     if (!found) {
     resultsContainer.innerHTML = "
Nenhum resultado encontrado.
";
     }
      }
